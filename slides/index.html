The Vue CLI

CLI Basics

  • Installed via NPM (or yarn)
  • Scaffolds application
  • Lets you add features
  • Serves a visual UI for managing your app
  • Prototyping
You may not now what NPM is - that's ok - I'm going to cover it in the next slide scaffolding is a fancy way of saying laying out all the files for you

WTF is NPM?

  • Node Package Manager
  • A versatile way to install things
  • Node developers use it, but you don't need to care about that
  • Install Node
I can create something that relies on other tools, A, B and C. I set things up so that my tool says it needs those things, and npm handles getting A, B, and C, and if A needs something else, it gets it, and if _that_ things needs something, it gets it, and so on.
